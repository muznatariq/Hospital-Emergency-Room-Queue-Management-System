# Hospital-Emergency-Room-Queue-Management-System 
Create a system that registers arriving customers or patients and issues them a token number. However, add a realistic twist: VIP customers or emergency medical cases must jump directly to the front of the line based on their severity.

This project is excellent for strengthening your grasp on Queues and Heap-based structures.

Data Structures to Use:
Priority Queue (using a Max-Heap or Min-Heap): A standard queue operates on FIFO (First In, First Out), but a Priority Queue allows you to sort elements dynamically so that the highest priority token is always processed next.
Circular Queue: To efficiently reuse token numbers once they are served and cleared from the system.
Why it matters: It teaches you how to handle resource allocation when data items do not have equal importance.
Language Used - Java
